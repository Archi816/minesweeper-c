# Miny — Terminal Minesweeper

A classic Minesweeper game running in the terminal, written in C.
First-year university project.

## Gameplay

- Reveal cells without hitting mines
- Flag suspected mines
- Clear the board to win

## Compile & Run

```bash
make
./build/mines
```

## Other commands

```bash
make test      # run unit tests
make coverage  # show code coverage
make clean     # remove compiled files
```

## Tech Stack

C · Terminal / CLI · GCC · Make
